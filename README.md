# SYNTHESIS Daily

Static public website for the daily "Synthesis delle ultime 24 ore" report.

The site contains no personal data. It is designed to be published with GitHub Pages and updated daily by Codex automation.

## Files

- `index.html`: current daily report.
- `archive/`: optional daily snapshots, one HTML file per date.

## Update model

Codex generates the daily report in Italian, covering relevant Technology and AI news found from consultable public sources in the previous 24 hours, deduplicated and aggregated.
